# 🔐 Password Manager

Простая программа для безопасного хранения паролей.

## 📥 Скачать

[![Скачать](https://img.shields.io/badge/Скачать-Password%20Manager%20-blue)](https://github.com/SenyaGeniy/Password-Manager-Windows/releases/tag/PasswordManager)

## 🚀 Как пользоваться

1. **Скачайте** файл `password_manager.exe`
2. **Запустите** его двойным кликом
3. При первом запуске **создайте мастер-пароль**
4. Добавляйте пароли через форму внизу
5. Используйте поиск для быстрого нахождения

## 📂 Где хранятся пароли?

Все пароли сохраняются в той же папке, где лежит программа:
- `passwords.json` - ваши пароли
- `master.key` - мастер-пароль

**Сделайте резервную копию этих файлов!**

## ⭐ Как создать ярлык на рабочем столе?

1. Нажмите на `password_manager.exe` **правой кнопкой мыши**
2. Выберите **"Отправить"**
3. Выберите **"Рабочий стол (создать ярлык)"**

Готово! Теперь можно запускать с рабочего стола.

## ⚠️ Важно

- **Не потеряйте мастер-пароль** — восстановить невозможно!
- Регулярно делайте резервную копию файлов `passwords.json` и `master.key`

Данное приложение — это **менеджер заметок с функцией сохранения паролей**,

**Автор НЕ гарантирует:**
- Абсолютную защиту от взлома
- Сохранность паролей при компрометации вашего устройства
- Восстановление утерянного мастер-пароля

**Вы принимаете все риски**, связанные с использованием программы.

Приложение использует базовое шифрование, но не является сертифицированным 
средством для хранения критически важной информации.
