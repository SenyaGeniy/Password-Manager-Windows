# 🔐 Password Manager Pro

Простая программа для безопасного хранения паролей.

## 📥 Скачать

Перейдите во вкладку **Releases** справа → скачайте `password_manager.exe`

## 🚀 Как пользоваться

1. Скачайте `password_manager.exe`
2. Запустите файл двойным кликом
3. При первом запуске создайте мастер-пароль
4. Добавляйте пароли через форму внизу
5. Используйте поиск для быстрого нахождения

## ✨ Возможности

- 🔒 Защита мастер-паролем
- 🎲 Генератор надежных паролей
- 🔍 Поиск по сайтам
- 📋 Копирование пароля в буфер
- 💾 Автосохранение

## ⭐ Как создать ярлык на рабочем столе

1. Нажмите на `password_manager.exe` правой кнопкой
2. Выберите "Отправить"
3. Выберите "Рабочий стол (создать ярлык)"

## ⚠️ Важно

- Не потеряйте мастер-пароль — восстановить невозможно!
- Пароли хранятся в папке с программой (файлы `passwords.json` и `master.key`)
- Делайте резервные копии этих файлов
